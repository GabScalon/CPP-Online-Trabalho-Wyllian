Sobre o projeto

O objetivo deste site é criar um editor de código em C++ online e, no processo, aprender mais sobre programação para web, realizando também o trabalho de projeto web para a matéria de Programação para Web (INE5646-03238A), lecionada pelo professor Wyllian Bezerra da Silva na Universidade Federal de Santa Catarina, no curso de Sistemas da Informação, utilizando as Linguagens HTLM5, CSS e JavaScript.
